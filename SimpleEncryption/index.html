Instructions

* Only the alphabet is available in this encryption.

* The button change, change between encrypt and decrypt options.

* The matrices used to encrypt and decrypt messages are the following.


Encoding matrix:
[1, -1, 2]
|-2, 0, 4|
[0, -2, 7]

Decoding matrix:
[4, 3/2, -2]
|7, 7/2, -4|
[2,  1 , -1]


Just messages encrypt with the encoding matrix can be decrypt in this page
